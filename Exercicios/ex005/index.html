TEGS E SEUS USOS

Textos Sobescrito

Para os texto sobescritos, ex: X20 + 3, com a teg sup.

Texto Subescritos

Para os textos subescritos, ex: H2O, usamos a teg sub.

Para códigos

Para códigos como pirnt('Olá mundo!'), nós usamos a teg code. Ele deixa oq esta dentro do code mais espaçado!

Códigos em pré definição

Usamos por ex: x = int(input(Digite um número: )) x1 = int(input(Digite outro número: )) print(x) print(x1) s = x1 + x print(f'A soma entre {x} e {x1} = {s}') print('Obrigado e volte sempre!')
Para deixar este código devidamente alinhado usamos a teg pre antes da teg do code. E ficará deste modo usando elas: 

            
        x = int(input(Digite um número: ))
        x1 = int(input(Digite outro número: ))
        print(x)
        print(x1)
        s = x1 + x
        print(f'A soma entre {x} e {x1} = {s}')
        print('Obrigado e volte sempre!')

Citações

Para usarmos uma citação como
Carlos eduardo disse:
Seja sempre uma luz para os outros
Usamos a teg q, para este tipo de citação.

Citações 2

Podemos usar também:

Estejamos sempre atentos para com os outros irmãos pela fé!

Deste modo a citação será deslocada para dentro do texto deixando de forma mais bonita! usamos a teg blockquote para essa função.

Marcações em cima do texto

Ao passsar o mouse em cima de algo como GC você terá a explicação do que é aquela abreviação ou palavra.
Para isso usamos a teg abbr.
